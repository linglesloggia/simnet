SimNet.extensions.simplesim

package documentation

A Simple SimNet simulator.

This simple simulation scenery is offered both as a test of the simulator framework and as an example of its extension, defining concrete classes inheriting from the abstract classes in the main library.

Module mk_simplescene Simple scenery with a base station, two slices, three user groups.
Module mk_simsetup SimNet template to setup a simulation scenery.
Module qa_interslicesched Inter slice scheduler test.
Module qa_mktrblk A test module for transport block making.
Module qa_oneusreq Test of reception and transmission with only one user equipment.
Module qa_pktqueue QA tests for the PacketQueue class.
Module qa_simulator Simple simulator tests.
