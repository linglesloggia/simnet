PyWiSim.extensions.simplesim.qa_simulator

module documentation

Simple simulator tests.

This script creates a simple simulation scenery, configs the entities attributes, and runs some tests. Beyond its testing purposes, this script and related module mk_simsetup.py provide a template to customize for different simulation sceneries: just copy these two files and change as desired.

Function test_assgn_res Changes resource assignment on the created scenery.
Function test_change_attrs Changes entities attributes on the created scenery.
Function test_mass_run Test with many users and long time, run.
Function test_mass_vrfy Test with many users and long time, verify with a short time.
Function test_move Runs a simulation with movement of user equipments.
Function test_run Runs a simulation on the simple simulation scenery.
Function test_show Create a simple simulation scenery and show its entities.
Variable ls_tests List of tests to show in menu.
